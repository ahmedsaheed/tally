# Tally Help

## Usage

`tally [directory | repository] [options]`

## Options

-   `-h`, `--help`

    -   Show this help message.

-   `-r`, `--remote [repository]`

    -   Tally a public repository.

-   `-b`, `--blame`
    -   List all of the files for each language.

-   `--html`
    -   Displays the reoport on the browser.
